# Gram
A statically typed functional language with graded modal types.

## Installation

To install using Stack:

    git clone https://github.com/dorchard/gram-language
    cd gram-language
    stack install

To install using only Cabal:

    git clone https://github.com/dorchard/gram-language
    cd gram-language
    cabal install
